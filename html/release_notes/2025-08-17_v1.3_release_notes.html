Release Notes - Version 1.3

  • Added a "Release Notes" section to the landing page with a modern, translucent popup for users to view release notes and documentation.
  • Implemented dynamic listing of release note HTML files, allowing easy addition and management of release notes.
  • Created a backend API endpoint to securely serve the list of release notes, improving compatibility and security.
  • Enabled static serving of HTML release note files for direct access from the popup.
  • Improved popup styling: enhanced transparency, consistent heading color, and better bullet alignment for document links.
  • Added support for grouping and managing release notes by version, with automatic cleanup of old files.
  • Documented the process for generating and maintaining release notes from git history.
  • General UI/UX improvements and bug fixes related to release notes display and navigation.
