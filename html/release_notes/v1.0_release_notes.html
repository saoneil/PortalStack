Release Notes - Version 1.0

Portal

  • Initial launch of the secure membership management portal.
  • User authentication, landing page, and basic navigation.
  • Consistent dark-themed UI and branding across all pages.

Patterns App

  • First deployment of the Patterns scoring app.
  • Core scoring and pattern management features.
  • Usability improvements: corner button size adjustments, reset all button label update.
  • Redeployment with NodeJS for improved performance and maintainability.

Sparring App

  • Initial release of the Sparring scoring app.
  • Full scoring functionality, including warnings and fouls.
  • Corner styling, color adjustments, and improved label visibility for better usability.
  • Landing page and controller formatting enhancements.
