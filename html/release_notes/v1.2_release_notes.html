Release Notes - Version 1.2

Portal

  • Release notes popup added for users to view app updates and documentation.
  • Backend improvements for dynamic release note management and static file serving.
  • UI/UX refinements for a more modern, user-friendly experience.

Patterns App

  • Total rework of the app, including pattern instance management, tie logic, and point aggregation for advanced scoring scenarios.
  • Added tie functionality for matches.
  • Enhanced score logic for black belts, master disconnect button, and logic for second patterns (yellow stripes).
  • Improved corner label formatting on the main scoreboard.

Sparring App

  • Major timer and match management upgrades: timer clock modifications for clarity, audible beep at end of timer, intermission timer, injury timer, and round counter.
  • Master disconnect button for quick control during events.
